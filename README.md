# 📄 PDF to Word Converter

A professional, lightweight tool that integrates a **"Convert to Word"** option directly into your Windows File Explorer right-click menu for PDF files.

![Modern UI](https://img.shields.io/badge/OS-Windows-blue?style=for-the-badge&logo=windows)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

## ✨ Features
- **Right-Click Integration**: Convert any PDF to Word without opening an application—just right-click in Explorer.
- **Silent Background Processing**: Conversion happens quickly and quietly.
- **Smart Completion**: Automatically asks if you want to open the converted Word document immediately.
- **Native Look & Feel**: Uses the official Microsoft Word icon in the context menu for a seamless experience.

## 🚀 Quick Setup (For Users)

If you just want to use the tool without messing with code:

1. Go to the [**Releases**](https://github.com/YousefBarakatt/PDFtoWord/releases) section of this repository.
2. Download the latest **`setup.exe`**.
3. Run the installer once.
4. **Done!** You can now right-click any `.pdf` file and select **"Convert to Word"**.

> [!TIP]
> The installer copies the necessary files to your `%LOCALAPPDATA%` folder and sets up the registry for you. You can delete the `setup.exe` file after installation is complete.

## 🛠️ Uninstallation

To remove the tool from your system:
1. Press `Win + R`, paste `%LOCALAPPDATA%\PDFtoWord` and hit Enter.
2. Run **`uninstall.exe`**.
3. The context menu entry and application files will be completely removed.
